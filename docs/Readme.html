LockWorkstation README


LockWorkstation provide a login screen to Haiku: With LockWorkstation you can protect and restrict accesses to Haiku. You can start it by Desktop icon (StartLock) and you can also add it to startup.

Please read this documentation to properly set and use LockWorkstation.

Usage

First of all is strongly recommended to change the default username and password (default username and password are baron - haikubox)


To change user and password go to Deskbar menu > Preferences > Login:


Then, in Login preflet you can change username and password and then clicking on "change" button:


Monitor resolution

Another important thing is to set the screen login according to your monitor resolution.
To know your current monitor resolution go to Deskbar menu > Preferences > Screen:


(in my case is set to 1280x1024: your resolution could be different)

Then open the folder /boot/apps/Lock_Workstation/images and you will see various subfolders with resolutions as name. Here select the folder which match to your monitor resolution, and edit his name to "default" (without quotes). If your monitor resolution is not listed, please mail me and i will provide to you ;-)


You're done. Optionally you can also set LockWorkstation to startup with Haiku.

Add to startup

Go to /boot/apps/Lock_Workstation and double click on "addtoboot" icon. You can also remove LockWorkstation from boot by double click on "removefromboot".


Rescue

If you forget your login details, don't you worry: i have added an embedded shortcut to close LockWorkstation without insert login details: just press CTRL+ALT+SPACEBAR. Then, to reset your password, go to /boot/home/config/settings and delete the file LockWorkstationSetting.
Then you need to reconfigure username and password and monitor resolution

Accesses report

Every time that you login, a logfile will be added to /boot/home/Login.txt
